# Titanic Dataset Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the famous Titanic dataset to understand survival patterns and key factors affecting passenger survival.

## Technologies Used
- Python
- Pandas (data manipulation)
- NumPy (numerical operations)
- Matplotlib & Seaborn (visualization)

## Key Insights
- Overall survival rate: 38.38%
- Female survival rate: 74.20%
- Male survival rate: 18.89%
- Higher survival rate in 1st class passengers
- Women and children were prioritized during evacuation

## Visualizations
1. Survival Count (Died vs Survived)
2. Survival by Gender
3. Age Distribution by Survival Status
4. Survival by Passenger Class
5. Correlation Heatmap
